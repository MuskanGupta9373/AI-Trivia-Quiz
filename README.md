# AI Trivia Challange

A simple console-based quiz app in C that tests your knowledge of Artificial Intelligence fundamentals....

## Features
- Timer to track quiz duration
- Score calculation and feedback
- AI-themed technical questions
- User-friendly console interface

## How to Run
1. Compile the code:
   `gcc ai_trivia_challenge.c -o ai_trivia_challenge`
2. Run the executable:
   `./ai_trivia_challenge` (Linux/macOS) or `ai_trivia_challene.exe` (Windows)

## Usage
Answer questions by typing the letter (A, B, C, or D) corresponding to your choice and pressing Enter.
